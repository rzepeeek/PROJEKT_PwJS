# Job Board

Projekt wykonany w ramach przedmiotu Programowanie w Językach Skryptowych.

## Uruchomienie projektu

Po pobraniu projektu należy zainstalować wymagane zależności:

npm install


Następnie uruchomić aplikację:

npm run dev


Po uruchomieniu w terminalu zostanie wyświetlony adres lokalny aplikacji, np.:

http://localhost:5174


## Wykorzystane technologie

* React
* Vite
* JavaScript
* Supabase
* React Router

## Funkcjonalności

* Rejestracja i logowanie użytkowników
* Obsługa ról: kandydat i rekruter
* Dodawanie ofert pracy
* Edycja i dezaktywacja ofert pracy
* Aplikowanie na oferty
* Zarządzanie aplikacjami kandydatów
* Dashboard użytkownika
* Integracja z bazą danych Supabase

Przed uruchomieniem należy uzupełnić dane połączenia z bazą Supabase w pliku src/services/supabase.js
